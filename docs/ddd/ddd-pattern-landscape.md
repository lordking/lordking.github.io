# DDD模式全景图

在DDD模式语言中，最为核心的概念
- 领域
- 通用语言
- 上下文
- 限界上下文
- 领域模型

### 支撑模式语言

为DDD模式语言扩展的DDD模式语言

### 通用模式语言
不属于DDD特有的模式语言，被用于支撑DDD模式语言实现的一般模式语言。这里重点介绍通用模式语言在DDD建模过程中的使用。
